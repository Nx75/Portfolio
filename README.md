## Color Palette (Gruvbox Dark)

| Variable       | Hex       | Usage                        |
|----------------|-----------|------------------------------|
| `--bg-hard`    | `#1d2021` | Primary background           |
| `--bg-dark`    | `#282828` | Cards, sections, nav         |
| `--bg`         | `#3c3836` | (optional) subtle surfaces   |
| `--fg-0`       | `#fbf1c7` | Headings, main titles        |
| `--fg-1`       | `#ebdbb2` | Body text                    |
| `--fg-2`       | `#d5c4a1` | Secondary / muted text       |
| `--aqua`       | `#8ec07c` | Links, primary accent        |
| `--yellow`     | `#fabd2f` | Highlights, warnings         |
| `--blue`       | `#83a598` | Info, secondary accent       |
| `--green`      | `#b8bb26` | Skill tags, success states   |


## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/Nx75/Portfolio.git
cd Portfolio

# 2. Choose one of the following ways to view:

# Option A – using Python (recommended)
python3 -m http.server 8000
# → open http://localhost:8000

# Option B – using Node.js (if you have http-server installed)
npx http-server -p 8080

# Option C – just double-click index.html (may have CORS issues with local images/scripts)
```
